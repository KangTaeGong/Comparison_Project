# Project Description

## SpringBoot Project - LeeSearch
Spring Boot를 이용한 영화 비교 사이트

## 💻 프로젝트 소개
**비교 검색 사이트**
검색어를 통해 영화를 검색, 비교할 수 있는 웹 사이트 입니다.

**제공하는 기능**
- 영화 제목 검색을 통한 정보 검색 및 두 영화의 정보 및 리뷰 비교.
- 자유 게시판
