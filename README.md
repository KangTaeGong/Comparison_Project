# SpringBoot Project - LeeSearch
Spring Boot를 이용한 영화 비교 사이트


# 프로젝트 소개
검색한 영화 제목을 통해 영화의 정보 및 리뷰 정보를 제공하고, 영화 두 개를 동시에 검색 시 두 영화의 정보를 동시에 보면서 비교가 가능한 사이트입니다.
Naver Movie API를 통해 영화의 정보를 가져오고 있습니다.
